# k8smap

See Makefile for all deployment options

## Usage

### Development
```
make node-depends

make node

make node-install-linux
# -or-
make node-install-osx

make node-run
```

### Docker

```
make docker
make docker-run

# debug..
make-docker-shell
```

## Quickstart

```
make node-linux
# -or-
make node-osx
```
